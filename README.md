# geosci-json

Geoscience JSON Building Blocks repository.

This repository uses the [OGC Building Blocks](https://github.com/opengeospatial/bblocks-postprocess) pattern to publish modular JSON Schema building blocks for geoscience metadata.

CI infrastructure and tooling are cloned from [CDIF/metadataBuildingBlocks](https://github.com/cdif-dt/metadataBuildingBlocks). Authoring guidance lives in [agents.md](agents.md); tooling docs in [USAGE.md](USAGE.md).

Building block sources go in `_sources/`. Build artifacts land in `build/` (generated; not committed in source PRs).
